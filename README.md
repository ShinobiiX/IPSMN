# IPSM Nigeria - Institute of Proficient Safety Measures

This is the official web portal for the **Institute of Proficient Safety Measures (IPSMN)**. Built with Next.js, it handles professional program applications, membership registrations, and provides information on industrial safety standards in Nigeria.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Components:** Shadcn UI
- **Icons:** Lucide React

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
